# xar.sh source

<https://xar.sh/>

Built with [Astro](https://astro.build/),
Served from [Cloudflare Workers](https://workers.cloudflare.com/).

```sh
npm run dev      # start dev server
npm run build    # build site + pagefind index
```

## Set envs

```sh
# set creds in `.mise.toml` and run:
mise set
```

## Generate image urls

```sh
npm run download
```
